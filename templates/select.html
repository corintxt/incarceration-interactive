The Incarceration Trends Project aims to inform the public dialogue, advance research, and help guide change by providing easily accessible information on jail and prison populations in every U.S. county.

The centerpiece of the project is an interactive data tool, available at trends.vera.org, that can be used for reference and measurement by justice system stakeholders and others looking to understand how their county uses jail and prison incarceration and how it compares to others over time.
