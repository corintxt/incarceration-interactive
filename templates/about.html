About

Incarceration.info provides an interactive set of charts and statistics to display nationwide data on incarceration collected by the Vera Institute of Justice.

An accompanying codebook gives more information on how the data was collected by Vera. The institute's own data exploration tool can be found at trends.vera.org.

This site is a collaboration between Corin Faife and Zoe Statman-Weil, and was built during a batch at the Recurse Center programming retreat in Brooklyn, NY. Charts are generated using the Altair visualization library.
